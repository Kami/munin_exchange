Welcome to Munin Exchange, a plugin repository for Munin inspired by http://www.nagiosexchange.org/

In order to participate on the new website (submit plugins and post comments) you don't need to create a new user account and you can use your existing OpenID account.

Quick start guide

  • If you want to upload plugins, create a user account by clicking "Register". The whole process is very simple, because you don't actually need to register and you only need a valid OpenID account.
  • For browsing and/or downloading, enter the search term in the input box in the left or go directly to the "Plugins" link.
  • If you want to upload a new version of an existing plugin, locate the plugin and click on the "Submit a new version" link on the plugin description page.
  • You also need to be registered to post comments
